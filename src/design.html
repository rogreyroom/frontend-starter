PROJECT SETUP

How to

The main setup is done in _vars.scss file.

Fonts

Add fonts for project, one for header and one for body, you can use @font-face or import. Next tweek fonts map to suit your needs - set font-familly, weights, headers and text sizes. In the fonts map is ms-base-ratio variable witch is use to setup base font size using modular scale and fluid-type mixin you can use modular scale to change it. The last thing is to setup media query sizes for minimum and maximum where the base font will be calculated between.

Colors

Setting up colors for the project is strait forward just change colors of the sass variables and you are done. In addition you can tweek percentage in collors map to setup shades for interactive color.

Media queries

In the mq map you are seting up media queries sizes you want to use in the project.

Dimensions

In the dimensions map you are seting up global dimensions for project/components eg. logo width, hamburger icon, form and any value you need to use globaly.


In addition values from color and dimensions map will be change to css variables in :root element as well as calculated base font.


It's all for the beer minimum - now start gulp dev or npm start and check design.html to see your changes


If you need to do more changes fell free to do so. My sugestion is to start with the commopnents - the rest is up to you :)
